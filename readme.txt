Create and showcase event timelines on your WordPress website available in multiple layouts with clean, elegant, and responsive design.

== Description ==

**Event Timeline Feed** plugin adds an event timeline view to your WordPress website. You can create multiple timelines for different pages of your site and add any number of events to the timeline with the event title, description, event link, icon, date, and time. Also, you can choose from multiple eye-catching layouts to showcase the timeline. Each layout has its unique sets of colors and typography settings, that you can use to further enhance the layout design.

== Installation ==

**Event Timeline Feed [Installation Guide]**

1. You can:
 * Upload the entire `timeline-feed` folder to the `/wp-content/plugins/` directory via FTP.
 * Upload the zip file of plugin via *Plugins -> Add New -> Upload* in your WordPress Admin Panel.
 * Search **Timeline Feed** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate the plugin through *Plugins* menu in WordPress Admin Panel.
3. Click on *Timeline Feed* menu to configure the plugin.
4. Ready, now you can use it.

== Frequently Asked Questions ==

= How to add timeline feed to a page or post? =
Use the shortcode **[timeline_feed id=ID]** (replace ID with the timeline ID) in any page or post to display the timeline feed.
